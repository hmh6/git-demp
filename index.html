我是首页 1.git init :初始化一个数据仓库 2.git status :查看状态 3.git add index.html :将index.html添加到暂存区 4.git commit -m 第一次提交 :将存在暂存区的数据传到本地仓库中。（第一次提交：是备注信息） 5.本地仓库准备好之后就可以在github中创建远程仓库 创建好之后在命令行中输入：git push https://github.com/hmh6/git-demp.git master (地址是从github中复制的、master是选择的分支) 因为上传的github地址太长，所以用：git remote add origan https://github.com/hmh6/git-demp.git 即可将长地址变为origan 在以后的使用中就会很方便。 从本地仓库传递到github的命令就会变为：git push origin master 在修改长地址变为origan之后还是觉得提交不方便。还可以继续进行修改
